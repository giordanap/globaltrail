# Changelog

All notable changes to GlobalTrail are documented here.

## 1.0.0 - 2026-05-10

### Release summary

GlobalTrail v1.0.0 is the first complete portfolio release of the travel intelligence dashboard.

This release includes country discovery, country detail, weather, currency, holidays, favorites, planner, notes, comparison, states gallery, Storybook documentation and quality passes for responsive UI, accessibility and performance.

```txt
- Premium home page

- Long weekends panel
- Persisted favorites
- Favorites page

- Travel planner page
- Destination comparison flow
- Compare destinations page
- Country travel notes
```
